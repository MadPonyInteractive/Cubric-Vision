# Cubric Studio v0.0.9 — 2026-06-10

## Changelog

Bug-fix patch, surfaced during the live macOS (Apple Silicon M4) acceptance test.

### Changes
- **Video zoom on macOS/Linux**: wheel-zoom, pan, and double-click-reset in the
  video viewer now work on macOS and Linux. The transform was previously applied
  to the player wrapper, which the hardware-video compositor on those platforms
  does not honour; it is now applied to the video element itself. (MPI-58)
- **macOS VAE precision**: ComfyUI now launches with `--fp32-vae` on Apple
  Silicon. fp16 VAE decode on MPS produced banded/corrupt output on an M4; running
  the VAE in fp32 fixes it at negligible cost.

### New Operations
None.

### Breaking Changes
None.

### ComfyUI Engine
ComfyUI version unchanged.
