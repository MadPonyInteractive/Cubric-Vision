# Cubric Vision v1.1.0 — 2026-07-09

Second minor release. Adds the RunPod remote engine, a full-page Model Library,
four new models (LTX 2.3, NVIDIA PiD Upscaler, Wan 2.2 5B, Chroma Flash), and a
large batch of workflow quality-of-life and fixes accumulated since 1.0.1.

## Changelog

### Important changes
- **Install individual model operations.** Multi-capability models (e.g. Wan 2.2
  does both text-to-video and image-to-video) now let you toggle which operations
  to download in the detail panel — shared parts (VAE, text encoder) download
  once. Only installed operations appear in the prompt box. Image models are
  unchanged.
- **Pick which GPU to install for.** Models with separate RTX 50-series vs
  40-series weights (e.g. LTX 2.3 Balanced) show a per-GPU toggle so you never
  fetch the wrong weight. Your GPU is pre-selected; a missing weight is offered on
  demand.
- **Removed the dead "ComfyUI API URL" field** (Settings → External Connections).
  It was never wired — generation always used the built-in engine. Pointing Cubric
  at a separately running ComfyUI instance is not supported.

### What's new
- **RunPod remote engine.** Run generation on a rented cloud GPU instead of your
  own machine — connect in Settings → RunPod and the app starts, installs models
  to, and generates on the Pod. Includes an optional minimum-system-RAM
  requirement and a live disk-usage bar showing used/total GB while connected.
- **Full-page Model Library.** Opening Models brings up a full-screen grid split
  into Installed/Available and Image/Video, with media/size/search filters, a
  per-model detail panel, a VRAM table, and fullscreen video previews.
- **New video model: LTX 2.3.** Fast video with sound — produce an audio track or
  drive it from your own clip. Text-to-video and image-to-video with
  first/last-frame guidance, up to 2K and 4K. Quality tier is remembered per model.
- **New model: NVIDIA PiD Upscaler.** A generative 4x image upscaler that adds
  real detail. Pick a look (Flux/SD3/Qwen/SDXL), output size (1K/2K/4K), and how
  much new detail to invent. Any aspect ratio.
- **New model: Wan 2.2 5B.** Fast low-tier video (text-to-video + image-to-video)
  in one compact 720p download, with a 4-step Turbo mode.
- **New model: Chroma Flash.** NSFW image model with exceptionally realistic skin
  (8-step Flux Schnell fine-tune), with standout matching upscale + detail passes.
- **Wan 2.2 Smooth quality boost.** Reworked two-stage sampling so the motion
  preview matches the final and stage-2 resolves more detail.
- **Searchable LoRA picker.** LoRA slots open a searchable, collapsible folder
  tree; long/deeply-nested names no longer clipped.
- **Model memory guidance.** Each model shows a size tier (Low/Balanced/High) with
  a hover table of the VRAM+RAM trade-off, your GPU row highlighted.
- **Drag-and-drop model import.** Drop a `.safetensors`/`.ckpt`/etc. onto a model
  folder in Settings and Cubric copies it in.
- **Model download queue.** Installing several models queues them one at a time,
  each cancellable, advancing on its own.
- **Prompt drafts survive navigation** (session-only, separate per surface).
- **Project right-click menu.** Right-click a project to add notes, rename it,
  open its folder, or clean up cached assets. Cleanup frees disk by removing the
  reuse frames saved for Reuse Prompt (generated media + history kept).
- **Reuse Prompt now covers video and audio**, not just images and settings.
  Toggle what carries over in Settings → Reuse Prompt.
- **Rename and annotate gallery cards.** Right-click a card to rename it or add
  notes; select multiple cards to copy them into another project at once. Renames
  stick across reloads (card, breadcrumb, prompt-box chip).
- **Multiple model folders.** Add more than one LoRA or upscale folder in Settings
  → External Connections. Extra folders are read-only; only your primary folder is
  managed for installs/removals.
- **Bypass a LoRA without removing it** (per-slot zero-strength toggle).
- **Hover to hear clips** (video/audio, one at a time; toggle "Play audio on
  hover" in Settings).
- **Smarter, configurable notifications** — fire whenever unfocused (incl.
  downloads), with independent generation/download toggles.
- **Video player & Cue QoL** — M to mute, Q for the Cue queue from a clip history,
  shortcuts on every player button hover.
- **Missing-model feedback** — missing LoRA blocks generation; missing upscale
  falls back to default with a warning.

### Fixes
- Reliable Stop (queued generations no longer finish anyway; status bar clears).
- Steadier stop/re-queue under load (queue rebuilt on one source of truth).
- Stop now works from a card prompt box.
- Subfolder LoRA/upscale models load on Windows (also the 1.0.1 hotfix).
- Universal upscale honors each model's native scale (1x/2x/8x).
- Relocated models no longer show as missing when found unambiguously elsewhere.
- Faster model installs (faster download network).
- Wan 2.2 LoRA settings drop the inert Clip slider.
- Cancelling a remote download clears leftover bytes immediately.
- Cue panel no longer covers the prompt bar.
- Downloads panel no longer flashes on refresh.
- Cards stop playing when scrolled off-screen.
- Wan 2.2 Smooth low-tier draft resolution raised.

### New Operations
- Text-to-video / image-to-video for the new video models (LTX 2.3, Wan 2.2 5B).
- Generative image upscale (NVIDIA PiD Upscaler).
- Chroma Flash image generation + matching upscale/detail passes.

### Breaking Changes
None. Project `SCHEMA_VERSION` unchanged (4).

### ComfyUI Engine
Updated to 0.27.0.
