## Step 2 — Find the layer list rendering

Look for where the tool builds the list of prompt layers — it likely renders a series of rows with drag handles.

Find the container element where layers are rendered (e.g., `#promptBuilder-layerList` or similar).

**Replace the layer list rendering with MpiDragList:**
```js
const layerList = MpiDragList.mount(
    container.querySelector('#promptBuilder-layerList'), // adjust ID to match actual template
    {
        items: currentLayers.map(layer => ({
            id: layer.id,
            label: layer.text || layer.label || '',
        })),
        draggable: true
    }
);

layerList.on('reorder', ({ items }) => {
    // items is the new order — update your layer array accordingly
    currentLayers = items.map(item => currentLayers.find(l => l.id === item.id));
    savePromptBuilderState();
});
```

> ⚠️ The actual layer data structure depends on the current `promptBuilder.js` implementation. Read it first, then adapt this to match the real layer object shape.
