## Step 3 ‚Äî Replace progress bar with `MpiProgressBar`

The current progress bar is raw DOM manipulation:
```js
const progressBar = container.querySelector('#upscaler-progressBar');
const progressWrapper = container.querySelector('#upscaler-progressWrapper');
// ...
progressWrapper.classList.remove('hide');
progressBar.style.width = '0%';
// and inside the callback:
progressBar.style.width = `${progress * 100}%`;
progressWrapper.classList.add('hide');
```

**Replace with:**
```js
import { MpiProgressBar } from '../components/Primitives/MpiProgressBar/MpiProgressBar.js';

// In initUpscaler(), after getting the container:
const progressWrapper = container.querySelector('#upscaler-progressWrapper');
const progressBar = MpiProgressBar.mount(progressWrapper, { value: 0 });
```

Then in `triggerUpscale()`:
- Replace `progressWrapper.classList.remove('hide')` ‚Üí `progressWrapper.classList.remove('hide'); progressBar.update({ value: 0 })`
- Replace `progressBar.style.width = \`${progress * 100}%\`` ‚Üí `progressBar.update({ value: progress * 100 })`
- Replace `progressWrapper.classList.add('hide')` ‚Üí `progressWrapper.classList.add('hide')`

> ‚öÝÔ∏è Check the actual `MpiProgressBar` props in `types.js` before writing ‚Äî use whatever prop it expects for the current value.
