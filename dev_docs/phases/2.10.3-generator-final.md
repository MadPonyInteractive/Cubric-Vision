## Step 1 — Verify `saveResultToLibrary` usage

Find where the generator saves the output image to the library.

It should look like:
```js
await saveResultToLibrary(imageUrl, 'generator');
Events.emit('media:updated', { projectId: state.currentProject?.id });
```

If it uses an old pattern (raw `fetch` to a save endpoint, or `document.dispatchEvent`), migrate it to `saveResultToLibrary` + `Events.emit`.
