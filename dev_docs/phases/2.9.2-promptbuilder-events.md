## Steps

### 1 — Replace `document.dispatchEvent` calls

Search for `document.dispatchEvent(new CustomEvent(` and replace each:
- `media:updated` → `Events.emit('media:updated', { projectId: state.currentProject?.id })`

### 2 — Replace `alert()` / `confirm()` calls

```js
import { showAlert, showConfirm } from '../dialogs.js';
```

- `alert(msg)` → `showAlert(msg)`
- `confirm(msg)` → `await showConfirm(msg)`

### 3 — Verify state persistence

Check that `saveToolState('promptBuilder', {...})` and `loadToolState('promptBuilder')` calls still exist and are unchanged. These must stay.

### 4 — Verify the `Events` import is at the top

```js
import { Events } from '../events.js';
```
