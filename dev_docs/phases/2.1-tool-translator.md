## Migration Checklist

### What to replace in `translator.js`

| Old pattern | New pattern |
|---|---|
| `import { PromptBox } from '../components/PromptBox.js'` | `import { MpiPromptBox } from '../components/Blocks/MpiPromptBox/MpiPromptBox.js'` |
| Manual DOM: `document.createElement`, `innerHTML =` | `MpiButton.mount()`, `MpiToast.mount()` |
| `document.dispatchEvent(new CustomEvent('media:updated'...))` | `Events.emit('media:updated', { projectId })` |
| Inline `debounce` implementations | `import { debounce } from '../utils/async.js'` |
| Direct `generateSeed()` from `uiHelpers` | `import { generateSeed } from '../utils/seed.js'` (if used) |

### What to keep as-is
- `import { state } from '../state.js'` — unchanged
- `import { saveToolState, loadToolState } from '../toolState.js'` — unchanged  
- `import { llmService } from '../llmService.js'` — unchanged
- `import { onLlmRunStart } from '../toolUtils.js'` — unchanged
- Global Ctrl+Enter in `init.js` — do NOT add keyboard handlers in this file
