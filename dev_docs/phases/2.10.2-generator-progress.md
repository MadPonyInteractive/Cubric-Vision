## Step 2 — Replace progress bar DOM

Find the raw progress bar code:
```js
progressBar.style.width = `${pct}%`;  // or similar
progressWrapper.classList.remove('hide');
progressWrapper.classList.add('hide');
```

Replace with `MpiProgressBar`:
```js
import { MpiProgressBar } from '../components/Primitives/MpiProgressBar/MpiProgressBar.js';

// During init, mount:
const progressBarComp = MpiProgressBar.mount(
    container.querySelector('#generator-progressWrapper'),  // adjust to actual ID
    { value: 0 }
);

// When starting run:
progressBarComp.update({ value: 0 });

// During progress callback:
progressBarComp.update({ value: progress * 100 });
```

> ⚠️ Keep the `progressWrapper.classList.remove/add('hide')` calls for show/hide — only replace the `style.width` manipulation.

## Step 3 — Replace `alert()` and `confirm()`

Add import if not already present:
```js
import { showAlert, showConfirm } from '../dialogs.js';
```

Replace all instances:
- `alert(msg)` → `showAlert(msg)`
- `confirm(msg)` → `await showConfirm(msg)` (make the wrapping function `async` if needed)

## Step 4 — Replace `document.dispatchEvent` calls

Search for `document.dispatchEvent(new CustomEvent(`. Replace each:
